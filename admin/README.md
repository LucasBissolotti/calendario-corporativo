# Instruções para Acessar o Painel de Administrador

O sistema possui uma conta fixa de administrador com as seguintes credenciais:

- **Nome de usuário**: admin
- **Email**: admin@admin.com
- **Senha**: admin123

## Como acessar o painel de administrador

1. Faça login com o nome de usuário "admin" ou email "admin@admin.com" e a senha "admin123"
2. Após entrar, você verá um link "Administração" no menu principal
3. Clique nesse link para acessar o painel de administração
